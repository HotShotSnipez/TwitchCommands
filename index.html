Twitch Commands

Twitch Commands
!lurk

Indicates that you are lurking in the stream without actively participating in the chat.

!iq

Displays an IQ score. You can also use !iq [username] to see the IQ score of a specific user.

Song Commands
!song

Provides details about the current song, including the song title, artist name, and a direct link to listen to it.

!next

Reveals the next song that will play in the queue.

!pos

Shows the position of your song request in the current queue.

Setup Commands
!pc

Lists the specifications and components of my current PC setup.

!setup

Details the gear and equipment used in my gaming setup.

Social Commands
!discord

Provides a link to join my Discord server where you can interact with the community.

!socials

Provides links to all my social media profiles.

Valorant Commands
!rank

Displays my current rank in Valorant.

!crosshair

Shows the crosshair settings I am using in Valorant.
